🔌 Enable OBS WebSocket

The plugin needs OBS WebSocket enabled to communicate with OBS.

How to Enable OBS WebSocket:

  1. Open OBS Studio
  2. Go to Tools → WebSocket Server Settings
  3. Check "Enable WebSocket Server"
  4. Make sure port is set to 4455
  5. Leave authentication disabled (unless you need it)
  6. Click OK
⚠️
Important!

OBS must be running for the plugin to work. The plugin will automatically connect when OBS is open.

Troubleshooting:

  • Make sure OBS Studio is running
  • Verify WebSocket is enabled in OBS (Tools → WebSocket Server Settings)
  • Check that port 4455 is not blocked by your firewall
  • Try restarting OBS Studio

🚀 Quick Start Guide

Here's how to use the main features:

  • 💡

    Scene Control with Hashtags

    Name your OBS scenes with hashtags like #center-stage or #pre-show. The plugin will automatically find and switch to these scenes.

  • 🎯

    Source Management

    Add #reset to source names to auto-hide them after 10 seconds. Add #ignore to sources you never want automatically controlled.

  • 💬

    Browser Sources for Overlays

    Create browser sources named #FakeChat #ignore and #Title-Reveal #ignore to use the Fake Chat and Title Reveal features.
